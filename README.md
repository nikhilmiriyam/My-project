# RideShare

RideShare is a car and seat booking system built with a simple MERN-style stack for local development.

## Project Overview
- Passenger and rider account flow
- Booking preview and confirmation flow
- Dashboard, account, history, sign-in, and sign-up pages
- Static frontend served by the Express server for easy local testing

## Setup
1. Install dependencies:
   - npm run install:all
2. Start the app:
   - npm run dev
3. Open the app in the browser:
   - http://localhost:5000

## Notes
- The backend runs on port 5000.
- The Vite client runs on port 3000 when started separately.
